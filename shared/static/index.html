Information/Support

Metamorphoun is a wallpaper changer that can change your wallpaper using pre-configured web sites or local picture folders. It can also display quotes of built-in selections or you can add your own using a json format like this:

[ {"statement":"Elections are a good deal like marriages, there`s no accounting for anyone`s taste.","author": "Will Rogers"}, {"statement":"Once a man wants to hold a Public Office, he is absolutely no good for honest work.","author": "Will Rogers"} ]

It defaults to a Christian Based Wallpaper Changer but it has enough configurations to suit anyone (with some work).

It is based on the original work of Peter Levi who wrote "Variety" in python for Linux. Now I follow him in Golang. I would like to thank him for his work and inspiration.

If you like this project and would like to support it, please consider donating to help keep the project going.
