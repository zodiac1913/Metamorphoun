Here is an example of a quote file in JSON format:

                                    
                                    [ 
                                        {"statement": "The only limit to our realization of tomorrow is our doubts of today.","author": "Franklin D. Roosevelt"},
                                        {"statement": "In the middle of every difficulty lies opportunity.","author": "Albert Einstein"}
                                    ]

If you name the file in a Pascal Case notation, (ie AlbertEinstein.json) the form will below will assume that is the author of the quotes and name the file accordingly.
